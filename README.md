# SK-Render-Skeleton

Python script that converts the skeleton or a set of joints in Autodesk Maya to set sphere and cone primitives substituting the wireframe standard mode of visualizing the skeletons in Autodesk Maya 2015. Using this tool, the skeleton can be rendered using any rendering engines such as Audodesk Maya or Mental-Ray rendering engines.
